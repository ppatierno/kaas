= Managing policies and permissions in Keycloak Authorization Services

:context: assembly-keyword

[role="_abstract"]
This section describes the relationship between Kafka's authorization model and the authorization model used in Keycloak Authorization Services.
It explains how to grant Kafka permissions to user accounts or service accounts using Keycloak Authorization Services rules.
Keycloak Authorization Services is part of Keycloak.
The mappings are used when granting permissions to access Kafka.

Examples are given showing the different user permissions required to perform common Kafka operations, such as creating and listing topics.
